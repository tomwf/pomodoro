# Pomodoro

A simple pomodoro app.

![](./screenshot.png)

## How To Use

```bash
git clone https://github.com/tomwf/pomodoro.git
```
Navigate into the directory:
```bash
cd pomodoro
```
Install the dependencies:
```bash
npm i
```
Run the local dev environment:
```bash
npm run dev
```
Open your browser and visit:
```
http://localhost:3000
```
